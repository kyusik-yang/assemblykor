# assemblykor - Handover (2026-03-11)

## Project Overview

**assemblykor** is an R package providing curated Korean National Assembly datasets
for teaching quantitative methods in political science. Korean politics counterpart
to `palmerpenguins`.

- **Version**: 0.1.0
- **Repository**: https://github.com/kyusik-yang/assemblykor
- **License**: MIT (code), data licenses vary (see README)
- **Local path**: `/Volumes/kyusik-ssd/kyusik-research/projects/assemblykor/`

## Package Contents

### 7 Built-in Datasets

| Dataset | Records | Scope |
|---------|---------|-------|
| `legislators` | 947 | 20-22nd assembly legislator metadata |
| `bills` | 60,925 | 20-22nd assembly bill metadata |
| `wealth` | 2,928 | Asset disclosure panel (13 time points) |
| `seminars` | 5,962 | Policy seminar activity panel |
| `speeches` | 15,843 | 22nd assembly Sci & ICT committee transcripts |
| `votes` | 7,997 | Plenary votes (20-22nd) |
| `roll_calls` | 368,210 | 22nd individual legislator votes |

### 7 Exported Functions

1. `get_bill_texts()` - Download bill rationale texts (60,925, ~40MB parquet)
2. `get_proposers()` - Download co-sponsorship records (769,773, ~25MB parquet)
3. `list_tutorials()` - List available tutorials
4. `run_tutorial(name)` - Run learnr interactive tutorial
5. `open_tutorial(name)` - Copy Rmd to working directory
6. `path_to_file(file)` - Get CSV file path
7. `set_ko_font()` - Auto-set Korean font for ggplot2

### 9 Tutorials (2 formats each)

| # | Topic | Key Methods | Datasets |
|---|-------|-------------|----------|
| 1 | R Basics and tidyverse | dplyr core functions, pipe, join | legislators, bills, wealth |
| 2 | Data Visualization with ggplot2 | bar, histogram, boxplot, scatter, line, facet | legislators, bills, wealth, seminars |
| 3 | Regression Analysis | OLS, multiple regression, log transform, interaction, coefficient plot | legislators, wealth, seminars |

**Two formats**:
- `inst/tutorials/` - learnr interactive (browser-based, English section headings)

## Technical Notes

- **CSS theming**: Custom dark theme (652 lines), Catppuccin Mocha inspired.
  Fonts: Inter (UI), JetBrains Mono (code). Uses `htmltools::includeCSS()` in
  each tutorial Rmd.
- **Data pipeline**: `data-raw/prepare_data.R` (R) + `process_*.py` (Python).
  Raw source files (.gitignored): `speeches_all_committees.csv` (3.1GB),
  `speeches_full.parquet` (1.4GB), `roll_calls_raw.csv` (44MB).
- **Dependencies**: `arrow` required only for download functions. Core package
  depends only on base R + utils.
- **LazyData**: Compressed with xz. Package install includes all 7 .rda files.
- **Arrow ALTVEC fix**: All 7 .rda files re-saved with forced `as.character()`
  conversion to prevent serialization errors from arrow's ALTVEC strings.

## Known Data Characteristics

- `n_bills` in `legislators`: includes co-sponsorship. Median 702, max 4,198.
- `n_bills_lead`: lead proposals only. Median 56, max 696.
- 661 unique legislators across 947 rows (some serve multiple assemblies).
- `wealth` unit: thousands of KRW (divide by 1,000,000 for billions).
- `roll_calls` API only available for 22nd assembly.
- `speeches`: 22nd assembly Sci & ICT committee only (15,843 records).
- Party colors: PPP (#E61E2B), DPK (#004EA2), RRP (#00B0F0).

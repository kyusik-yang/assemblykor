## Datasets

Seven built-in datasets covering legislators, bills, asset declarations,
policy seminars, committee speeches, plenary votes, and individual roll
call records.

- [`legislators`](https://kyusik-yang.github.io/assemblykor/reference/legislators.md)
  : Members of the Korean National Assembly (20th-22nd)
- [`bills`](https://kyusik-yang.github.io/assemblykor/reference/bills.md)
  : Bills Proposed in the Korean National Assembly (20th-22nd)
- [`wealth`](https://kyusik-yang.github.io/assemblykor/reference/wealth.md)
  : Legislator Asset Declarations (2015-2025)
- [`seminars`](https://kyusik-yang.github.io/assemblykor/reference/seminars.md)
  : Policy Seminar Activity by Legislator-Year (2000-2025)
- [`speeches`](https://kyusik-yang.github.io/assemblykor/reference/speeches.md)
  : Committee Speeches from the Science and ICT Committee (22nd
  Assembly)
- [`votes`](https://kyusik-yang.github.io/assemblykor/reference/votes.md)
  : Plenary Vote Results in the Korean National Assembly (20th-22nd)
- [`roll_calls`](https://kyusik-yang.github.io/assemblykor/reference/roll_calls.md)
  : Member-Level Roll Call Votes (22nd Assembly)

## Download Functions

Download larger datasets not included in the package binary. Results are
cached locally after the first download.

- [`get_bill_texts()`](https://kyusik-yang.github.io/assemblykor/reference/get_bill_texts.md)
  : Download bill propose-reason texts
- [`get_proposers()`](https://kyusik-yang.github.io/assemblykor/reference/get_proposers.md)
  : Download bill co-sponsorship records

## Tutorials

Nine Korean-language tutorials for classroom use, available in both
interactive (learnr) and plain R Markdown formats.

- [`list_tutorials()`](https://kyusik-yang.github.io/assemblykor/reference/list_tutorials.md)
  : List available tutorials
- [`open_tutorial()`](https://kyusik-yang.github.io/assemblykor/reference/open_tutorial.md)
  : Open a tutorial file
- [`run_tutorial()`](https://kyusik-yang.github.io/assemblykor/reference/run_tutorial.md)
  : Run an interactive tutorial
